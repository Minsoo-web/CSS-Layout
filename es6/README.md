## 🚀 Intro

ES6는 ECMAScript가 2015년에 발표한 표준 자바스크립트 6번째 에디션입니다.  
ES7, 8 등등 많은 버전들이 있지만 ES6를 제일 많이 들어보셨을겁니다.  
그 이유는 ES5에서의 JS 문법과 ES6가 도입된 JS 문법은 너무 많은 차이가 있기 때문입니다.
